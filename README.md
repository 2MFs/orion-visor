**`orion-visor`** 提供一站式服务器运维解决方案。

* **资产管理**：支持对资产进行分组，实现对主机、密钥和身份的统一管理和授权。
* **在线终端**：提供在线 Terminal 服务，支持快捷命令、大文件上传、在线编辑等操作。
* **批量操作**：支持批量执行主机命令、多主机文件分发等功能。
* **计划任务**：支持配置 cron 表达式，定时执行主机命令。
* **安全可靠**：动态配置权限，记录用户操作日志，提供简单的审计功能。
